# LAB-Keylogger
Criando e executando um keylogger 

Neste lab foi criado um keylogger que é executado em segundo plano.
Toda vez que o usuario digita uma tecla o programa captura a mesma.
Tudo que for digitado será gravado em um arquivo .txt que mostará tudo de forma sequencial

A proxima etapa foi com o intuito de deixa-lo ivisivel para o usuário
Onde foi usado o comando ren .\keylogger.py .\keylogger.pyw para a mudar a extensão

Por fim, o ultimo script foi criado para enviar os dados capturados para o atacante
